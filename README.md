VIDEO GAME API
----------------------------------------------------------------------------
localhost:8084 is the default way to access the api

Common Usage

  GET
    /users
      Gets all users
    /users/{userId}
      Gets a user specified by the userId
    /games
      Gets all games
    /games/{gameId}
      Gets a game specified by the gameId
    /offers
      Gets all offers
    /offers/{offerId}
      Gets an offer specified by the offerId
